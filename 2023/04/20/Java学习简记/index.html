Java 学习简记

静态内容初识

  • 静态内容是属于类的，它会先于类的实例化（对象的生成）建立。拥有较高的优先级。但是，如果类并没有初始化（赋值，new 构造，调用方法）则出于 Java 的优化，静态变量不会被创建。

Java 中的代码执行顺序

  • 静态变量 > 静态代码块 > 静态构造函数 > 成员变量 > 代码块 > 构造函数
